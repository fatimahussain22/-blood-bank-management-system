# Blood Bank Management System

A full-stack web application for managing blood donors, built as part of an Advanced Database Systems lab project at IoBM.

## Features
- Admin dashboard to manage and view all donors
- User registration and login system
- Add, view, and search blood donors by blood group
- Secure password hashing with bcrypt

## Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas, Mongoose
- **Frontend:** HTML, CSS, JavaScript

## How to Run
1. Clone the repo
2. Run `npm install`
3. Start the server with `node server.js`
4. Open `frontend/index.html` in your browser
